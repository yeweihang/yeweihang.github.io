CSS选择器

1、基础选择器

  • 标签选择器

    标签选择器的特性：它选择的是某一类的标签，也就是说，选择固定区域的所有的这一类的标签。

  • id选择器

    id选择器的特性：选择独一无二的某一个标签
    注：

    • id名是独一无二的，在页面当中不可能出现两个相同的标签；
    • 遵循标识符规则；
    • id选择器在制作静态页面的时候，尽量少用。

  • 类选择器

    • 同一个标签可以拥有不同类；
    • 不同的标签可以拥有相同的类；
    • 类选择器用的好坏，可以看出你前端的经验(抽象公共类)。在实际开发中，拿到效果图，先分析效果图，看看是否有公共类，先去抽象公共类

高级选择器

  • 通配符选择器

    这个标签通常在教学中用来CSS初始化，选择固定区域内所有的标签进行样式设置。

  • 并集选择器

    • 选择器1选择到的标签和选择器2选择到的标签如果拥有相同的样式，我就可以用并集选择器。(并且的意思)
    • 并集选择器也被常用于CSS初始化，它和*都能对CSS初始化，但是并集选择器出现化效率比*高，因为*选择的是所有，有的没的都找一遍，而并集选择器只把自己用的初始化。
  • 交集选择器

选择器1所选择的标签和选择器2选择到的标签相交的部分，设置样式。

  • 后代选择器

选择的是选择器所选的标签的后代里面符合选择器2的标签
后代选择器用法
1、搞明白自己要选择谁
2、唯一的去确定他们的祖先
3、通过这个祖先运用后代选择器

  • 子代选择器

选择的是选择器1 所选的标签的子代(仅仅是儿子辈)里面符合选择器2的标签。
