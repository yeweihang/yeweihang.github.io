javascript基本编码

js数据类型

定义变量关键字 var
js的原始类型

  • string：字符串

    var str = “abc”;

  • number:数字类型

    var m = 123;

  • boolean:true和false

    var bl = true;

  • null

    var date = new Date();
    获取对象的引用，null表示对象的引用为空，所有对象的引用也是object

  • undifined

    定义一个变量，没有赋值
    var aa;

typeof() 可以查看变量类型

js语句

和Java没什么区别

  • for
  • while
  • switch

    可以支持javascript的所有类型

js运算符

几个特别之处

  • 在js里面不区分整数和小数，123/1000=0.123,0.123*1000=123

  • 字符串的相加减

    相加和Java没有什么区别
    相减

    • 在Java里这是不允许的；
    • 在js里执行相减的时候，执行减法运算；
    • 但是如果字符串不是一个数字，如str = “abc”就会提示NaN

  • 布尔类型的相加减运算

    true的值为1；
    false的值为0；

  • ==和===的区别

    ==是比较值;
    ===是比较值和类型;

js数组

数组的遍历

js方法

不需要声明返回值类型；
函数的参数列表不需要写var,直接写变量;
不要忘记调用；

js全局变量和局部变量

全局变量：

在script标签里面定义一个变量，这个变量在页面中js部分都可以使用
在方法的外部可以使用，在方法的内部可以使用，在另一个script标签中可以使用

局部变量：

在方法的内部定义一个变量，只能在方法的内部使用
如果在方法的外部调用这个变量，提示出错
Uncaught ReferenceError: nn is not defined

  • 直接添加在html中

    在body里或在的下面

  • 写在js文件中

    使用<script type="text/javascript" src="js/html.js">引入
