一、CSS的简介

1、css是什么？

css是英文cascading style sheet的缩写，就是层叠样式表。

2、css的作用是什么

给网页结构的标签设置样式使得网页看起来美观。

3、怎么去用css

css的组成：

选择器+属性

css的属性：

  • 文本类

    font-family———-字体
    font-size————字体大小
    font-weight———-字体的粗细
    font-style———–字体格式
    color—————-字体颜色
    text-align———–对齐方式
    text-decoration——下划线
    line-height———-设置行高，可以控制文本的行间距
    vertical-align——-垂直对齐方式(只能针对行内元素设置)

  • 背景类

    background-color—–设置背景颜色
    background-image—–设置背景图片

  • 位置类

    盒子模型
    浮动float
    定位posion

二、颜色和大小的设置方式

1、颜色的设置

  • 英文单词设置，但是这种设置在实际开发项目中用的很少，因为单词很局限。
  • rgb三原色十进制设置rgb(255, 255, 255);rgb分别代表红绿蓝，他们每种颜色的值，可以是0~255.
  • rgb三原色十六进制设置#fff，十六进制是通过三原色的十进制转换而来的，在实际开发当中用的最多，因为它可以表示所有的的颜色，并且设置简单。
  • rgba它是带有透明度的三原色设置rgba(255, 255, 255, 0.3)，a代表透明度，它的值是在0~1之间，0代表全透明，1代表不透明，中间值代表半透明。

2、大小的设置

  • px，是像素，它是所有大小单位的本质。
  • 百分比，是相对设置元素的爹去做百分比。

  • em，也是被设置的元素，相对他爹进行设置，但是这个em值是一直变化的，也就是说，爹的字体有多大，儿子里面1em就有多大。

    div中有个儿子p，p也有一个儿子span.
    div设置字体大小20px,p设置字体大小2em，span设置字体大小1em，那么p里面的em代表20px，而span里面的em代表40px。

  • rem，它是根据em改良而来的，em用起来比较混乱。所以出现的rem，rem是相对根元素的字体大小进行设置。它和爹没有半毛钱关系。
