javaScript的DOM

发表于 2017-08-09   |   分类于 javaScriptNote   |   评论

基础对象

js的dom对象

  • dom: document object model: 文档对象模型
    • 文档：超文本标记文档。html、xml
    • 对象：提供属性和方法
    • 模型：使用属性和方法操作超文本标记型文档
      • document对象
      • element对象
      • 属性对象
      • 文本对象
      • Node节点对象：这是上面对象的父对象；

DHTML

是很多技术的简称

  • html:封装数据
  • css：使用属性和属性值设置样式
  • dom：操作html文档
  • javascript：专门指的是js的语法语句（ECMAScript）

document对象

表示整个文档

常用方法

  • write();

    向页面输出变量
    向页面输出HTML代码

  • getElementById();

    通过Id来得到元素，返回的是对拥有指定id的第一个对象的引用；

  • getElentsByName();

    通过标签的name值得到元素，返回的是指定名称的对象的集合；

  • getElementsByTagName();

    通过标签来得到元素，返回指定标签的对象的集合；

注意：
只有一个标签， 这个标签只能使用name获取得到，这个使用getElementsByName返回的是一个数组，但是现在只有一个元素，这个时候不需要遍历，而是可以直接通过数组的下标获取值

案例：widows弹窗案例
1、创建一个页面

- 有两个输入项和一个按钮
- 按钮上面有一个事件：弹出一个窗口 open

2、创建弹出页面

- 表格
- 每一行有一个按钮和编号和名字
- 按扭上有个事件：把当前的编号和名字，赋值都第一个页面相应的位置

案例：在末尾添加节点

  • 第一步：获取ul标签
  • 第二步：创建li标签
    createElement(标签名);
  • 第三步：创建文本
    createTextNode(“字符串”);
  • 第四步：把文本添加到li下面
    appendChild(对象名);
  • 第五步：把里添加到ul末尾

Element对象

要操作element对象，首先要获取到element。(使用document里面相应的方法获取)

方法

  • getAttribute();获取属性
  • setAttribute(属性,属性值);设置属性值
  • removeAttribute(属性);删除属性。是不能删除value属性

想要获取标签下的子标签

  • 使用属性childNodes,但是这个属性的兼容性很差
  • 获取标签下的子标签的唯一有效方法，使用getElementsByTagName方法

Node对象

属性一

  • nodeName
  • NameType
  • nodeValue

使用dom解析html时候，需要html里面的标签，属性和文本都封装成对象

标签节点对应的值

  • nodeName：1
  • NameType：SPAN
  • nodeValue：null

属性节点对应的值

  • nodeName：2
  • NameType：属性名称
  • nodeValue：属性值

文本节点对应的值

  • nodeName：3
  • NameType：#text
  • nodeValue：文本值

属性二

父节点

parentNode
ul是li的父节点

子节点

li是ul的子节点
childNode:得到所有的子节点，兼容性差
fristChild:获取第一个节点
lastChild:获取最后一个节点

同辈节点

li之间是同辈节点
nextSibling:返回一个给定节点的下一个兄弟节点
previousSibling:返回一个给定节点的上一个兄弟节点

操作DOM树

  • appendChild方法

    • 添加子节点到末尾
    • 特点：类似于剪切粘贴的作用

  • insertBefore(newNode,oldNode)方法

    • 在某个节点之前插入一个新的节点
    • 两个参数
      • 1、要插入的节点
      • 2、在谁之前插入
    • 插入一个节点，节点不存在，
      • 1、创建标签
      • 2、创建文本
      • 3、把文本添加到标签下面

  • removeChild();删除节点

    • 通过父节点删除

  • replaceChild(newNode,oldNode);替换节点

    • 不能自己替换自己，通过父节点替换
    • 两个参数
      • 1、第一个参数：新的节点
      • 2、第二个参数：旧的节点

  • cloneNode(boolean);复制节点

innerHTML属性

不是DOM的组成部分，但是大多数的浏览器都支持

  • 第一个作用：获取文本内容
  • 第二个作用：向标签里面来设置内容(可以是HTML代码)
