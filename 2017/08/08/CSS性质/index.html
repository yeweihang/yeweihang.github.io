CSS性质

1、继承性

color text font line，以这个四个单词开头的属性，都具有继承性。

  • 什么是继承性

    如果一个元素作为祖先，它设置了上述的四种属性，，那么他的后代全部都会具有上述属性。但是中间不能有哪一代造反打断，如果继承性被中间的某一代打断，那么这一代的后代的后代将会继承打断的这一代。

2、层叠性

  • 当一个元素同时被多个选择器选中的时候，并且这多个选择器给这个元素添加不同的属性样式，那么所有的这些属性都会叠加到这个元素上。
  • 当一个元素同时被多个选择器选中的时候，并且这多个选择器给这个元素添加相同同的属性样式，那么要遵循以下规则：
    • 权值问题：

      id 100
      class 10
      标签 1
      行内样式 1000
      权值的大小，决定优先级的先后。谁的权值大，代表谁的有限级高，那么被选中的元素将会听这个有限级高的选择器，如果权值的大小一样，谁写在后面听谁的；如果这个元素存在行内样式，那么就按照行内样式设置了。

    • 如果多个选择器选择相同的标签，但是其中部分选择器最终没有选择到这个标签，那么对于这个标签来讲，这些没有选择到的选择器权值为0；如果选择器没有选择到就看继承性。

四、标准文档流

1、什么是标准文档流

浏览器在解析我们文档的时候，从左到右，从上到下去解析的这样一个过程，形成一个流，被形象的称为标准文档流。标准文档流我们理解为在同一个平面上。

2、标准文档流中对标签的分类

  • HTML分类

    容器级别：h1-h6 ul ol li dl dt dd table div
    文本级别：p img input select textarea label span

  • 标准文档流分类

    块级标签：容器级别、p
    行内标签：文本级别中除了p

3、块级标签和行内标签

  • 块级标签：

    它很霸道，无论设不设置宽，它都要占据它爹的一行，块级便签如果不设置宽，那么它的宽将是爹的宽，如果不设置高，那么它的高度将是内容撑开的高。块级标签如果设置了宽高，但是它还是会占据爹的一行。

  • 行内标签：

    它很懦弱，如果是多个标签，那么他们会并排去排在一行，只有当这一行容不下的时候，才会换行。行内标签如果不设置宽高，那么它的宽和高有内容决定。行内标签如果设置宽高，能设置，但是不理会。也就是说行内标签它的宽和高只能由内容决定。

行块转换

  • display: block; 代表盒子的表现形式为块级的；
  • display: inline; 代表盒子的表现形为行内的；
  • display: none; 代表盒子的表现形式为隐藏；

    如果想让一个块级标签转换为行内标签，给它添加display: inline;属性。
    如果想让一个行内标签转换为块级标签，给它添加display: block;属性。
