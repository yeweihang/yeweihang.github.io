HTML概述

发表于 2017-08-08   |   分类于 HTMLNote   |   评论

一、什么是HTML

HTML文件本质是一个纯文本文件，但是它通常称为超文本文件。
HTML语言：它是一种编程用的标记语言，用它编写的文件是HTML语言。hypertext markup language 。

  • 1、什么是纯文本文件

文件名由两部分构成：文件名+后缀名，文件的后缀名一般可以告诉我们它是一种什么文件。我们通常见到的txt它就是一种纯文本文件，纯文本文件但当只能编辑文字文本信息。

  • 2、为什么浏览器默认渲染的文件是html文件而不是纯文本文件

HTML虽然是纯文本文件，但是它被称为超文本文件，他不但可以编辑文本信息，还可以通过一些手段进行链入图片、音乐等等的信息而纯文本文件是做不到这一点的，因此浏览器最终选择了HTML文件作为渲染文件。

  • 3、浏览器在打开纯文本文件和HTML文件的时候，这些文件都是浏览模式。但是同样的文章，不同的形式，为什么？

HTML文件当中文字生来平等，没谁就是标题。谁就是段落。我们需要一种手段让浏览器知道谁是标题谁是段落，这时候需要一种手段，而这个手段就是HTML语言，让浏览器知道谁是标题谁是段落，
HTML：它的作用就是给文本信息添加语义的，除了语义什么都没有！（<h2>就是一个标题，并不能改变文字的样式）
