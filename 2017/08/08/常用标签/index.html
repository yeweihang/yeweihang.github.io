常用标签

发表于 2017-08-08   |   分类于 HTMLNote   |   评论

一、常用标签

从语义的角度去描述结构 HTML
从审美的角度去描述样式 CSS
从交互的角度去描述效果 JavaScript

1、标题标签

h${我爱你}*6-----生成h1到h6------($是递增)
  • h标签是英文heading的意思，标题的意思。
  • 它是一个家族标签，从h1到h6；
  • 它的作用：给文本添加语义
  • 它是一个容器级别的双标签。

2、段落标签

  • p标签是英文paragraph的意思，段落的意思。
  • 它是用来给文本添加段落语义的。
  • p标签是文本级别的双标签。

容器级别的标签，内部什么都可以放，可以放文本，也可以放文本级别的标签，还可以放容器级别的标签。
文本级别的标签，内部可以放文本，也可放文本级别的标签。但是不能放容器级别的标签。
(如果将h便签放到p标签里，p标签会将h踢出去;<p><h2></h2></p> ——> <p></p><h2></h2><p></p>)

3、图形图像

  • img标签，它是英文image的意思，就是图片的意思，在这我们用来添加或者引入一张图片用的。
  • 它是一个文本级别的单标签。
  • 它里面有两个属性要记住：
    • 1、src属性，它是用来添加图片，src的值是图片的相对路径或者绝对路径，和href类似。
    • 2、alt属性，当图片因为某种原因，加载失败的时候，可以用alt添加替换文本。

4、关于文本处理的几个文本级别的双标签

  • 1、引用语义
    • q标签用来对文本的一句话，或者文本的部分内容添加引用语义。
    • blockquote标签用来对整段文本添加引用语义。
  • 2、强调语句
    • em标签用来对文本进行轻微强调。
    • strong标签用来对文本进行着重强调。
  • 3、
    • br用来对文本强制换行
    • hr用来给文本添加分割线

5、超级链接标签

  • a标签，a的英文是anchor的缩写，锚的意思。是一个文本级别的双便签用来给网页添加超级链接。
  • 有三个属性，我们要记住
    • href，它是用来填写目标的地址，可以是相对路径也可以是绝对路径
    • target，这个属性，决定着目标地址是覆盖还是新建窗口打开。
      • _self 覆盖自己(默认)
      • _blank 新建窗口打开
    • title，这个属性是个客户看的，它里面填写的是点击后去的目的地。告诉客户，这个超级链接是链向什么地方的，这个属性根据需要添加，可有可无。
  • 超级链接的常用场合
    ①以一句话为点击内容
    ②以段落为点击内容(a便签放在p中)
    ③以图片为点击内容
    ④用来制作站内锚点
    • 它需要两个a标签配合使用。目的地a需要一个id名。
    • 点击中的a标签中的href中填写的就是目的地的a标签的id名

二、三种列表

无序列表 有序列表 自定义列表

  • 无序列表

    • ul是英文Unorder list的缩写，无序列表的意思。它可以在页面上添加列表的语义。ul是我们见到的第一个组标签，他需要和li配合使用，缺一不可。li是英文list item的缩写。ul标签中除了li什么都不可以放，li当中什么都可以放。
    • ul li都是容器级别的双标签。

  • 有序列表

    • ol是英文order list的缩写，有序列表的意思。它可以在页面上添加列表的语义。ol是我们见到的第一个组标签，他需要和li配合使用，缺一不可。li是英文list item的缩写。ol标签中除了li什么都不可以放。li当中什么都可以放。
    • ol li都是容器级别的双标签。

  • 自定义列表

    • dl是自定义的意思，英文definition list的缩写、他也是一种列表，组标签，在dl当中至少出现一个dt。dt是definition title是自定义主题的意思。每个dt可以有多个dd对它进行描述。dd是definition description的缩写，自定义的意思。
    • dl dt dd都是容器级别的标签
    • 一般采用一个dl中一个dt，适合布局

三、表单类元素

1、5种输入框

input 标签，是文本级别单标签这个标签很nb,它根据type属性的不同，表现出不出来的样子不同。

①、文本输入框——type值为text

placeholder属性是可以设置文本输入框当中的提示语。
value属性代表的是文本框当中的输入的内容，可以通过设置value属性中，设置文本框当中的内容，value属性出现就代表设置，不出现就它就是用来存储输入的内容。

②、密码输入框——type值为password

placeholder属性是可以设置密码输入框当中的提示语。
value属性一般在密码输入框中不出现，不过value属性在数据交换中也会使用。

③、单选输入框——type值为radio

单选输入框都是成组出现的，因此name属性我们要牢记，相同的name值，代表的是同一组。在同一组当中，单选框只能同时选一个。
无论是id名还是name名，在取名的时候全部要遵从标识符规则。
checked=”checked”这个属性，可以设置默认选项是哪一个。

单选框的绑定

label

label是一个无语义的标签，它通常在表单类元素里面出现，用来绑定文本和输入框，他还可以用来放置文本，防止文本在body中裸奔。

④、多选输入框——type值为checkbox

⑤、日期输入框——type值为date 兼容性有问题，IE不兼容

⑥、三种按钮

2、下拉列表

select也是文本级双标签，只能放option。

3、文本域

文本域txetarea也是一个文本级别的双标签。

4、表格table

table是容器级别的双标签

5、div和span

div标签，它是divition的缩写，分割切割的意思。我们通常把他称作盒子，他没有语义，但是用的很多，主要用于对网页进行分块。它是容器级别的双标签。
span标签，它也有分割切割的意思，它也用来分割网页，与div不同的是，它是文本级别的双标签。

四、常用标签总结

容器级别：h1-h6 ul ol li dl dt dd table div
文本级别：p img a input select textarea span
所有的标签都是盒子。只不过有些盒子有语义，有些没有语义罢了。
