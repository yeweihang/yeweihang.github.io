引用类型

引用类型的变量只是保存真实数据的地址，而不是数据本身。引用类型只给变量分配引用空间，数据空间是没有分配的，因为谁也不知道数据是什么啊，整数，字符？引用类型变量在申明后必须通过实例化开辟数据空间，才能对变量所指的对象进行访问。

引用只能指向一个系统创建的对象，就不同与指针，指针可以修改指针的值，把指针指向任何地方，会造成安全问题，而引用就不会。

系统强制转换

不同的类型的数字进行运算时，系统会强制改变数据类型。

在程序编译时上面的代码是会有Error：
Type misamtch:cannot convert from int to byte

因为在执行a+b时，系统会把a和b的类型转换为int类型然后计算，计算的结果也是int型，所以在把int类型赋值给byte类型时就产生了错误。

系统强制类型转换有如下的规则

  • 操作数如果有double类型，则都会转换成double类型。
  • 如果有float类型，则会转换为float类型。
  • 如果有long类型，则都会转换为long类型。
  • 其他的都会转换为int类型。

&&与&的区别和||与|的区别

运行结果

可以看出 && 和 & 的运算结果是相同的，单b和c的值不同，使用 && 时，后面的表达式没有计算；使用 & 时，后面的表达式进行了计算。实际上是 && 运算前面的表达式为false，结果就是false，所以后面就不用计算，但如果为true，后面的表达式就要进行运算。
|| 与 | 的区别也是这样。
【记住】 && 和 || 是快速运算符，但不能保证后面的表达式会被执行。

三目运算符

a?b:c;

a的结果应该为布尔类型，如果a的值为true，则结果为b;如果a的值为false，则结果为b。
