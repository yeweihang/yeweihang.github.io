浮动

1、什么是浮动
浮动就是CSS当中的一个属性，会使元素向左或向右移动，其周围的元素也会重新排列

2、浮动的作用
浮动就是布局网页的主要手段，它可以让块级的标签并排
3、怎么运用浮动
浮动的元素没有行块之分；所有的元素，都可以设置宽高，并且所有的元素如果没有设置宽，那么他的宽就是内容的宽，
浮动的元素不能使用margin: 0 auto;让盒子居中；
浮动的元素兄弟垂直方向上margin不塌陷

  • 浮动的性质
    • 浮动具有脱标性

      脱标指的是脱离标准文档流，在我们学习过程中会碰到三种情况会脱离标准文档流，还有绝对定位和固定定位
      浮动描述的一个家族悲惨的血泪史，我们在初学浮动的时候，记住浮动说的是儿子，儿子要浮动全浮动，如果不浮动，就会被浮动的兄弟遮住；爹一定要有高度，如果爹没高度，就会死，爹死了会造成很大影响，网页无法布局。

    • 浮动的元素具有紧贴性(了解)

      浮动的元素都有紧贴性的性质，当爹的一行无法装下所有的兄弟，最后一个元素会被挤下来，但是它会去找兄弟去紧贴，如果看不到所有的兄弟就会去贴爹。

    • 浮动的元素具有字围效果

      当一个盒子浮动，它的兄弟不浮动，那么兄弟的盒子会钻到浮动盒子的下面，但它里面的文字内容不会钻到浮动盒子的下面，它会围绕着浮动的盒子。

4、清除浮动
可以对任意的盒子清除浮动。清除这个盒子周围浮动的元素，也就是说这个盒子周围不能有浮动的元素。

  • 清除浮动的方式1
    给爹添加高度，爹有高度，它的儿子怎么浮动都可以。
  • 清除浮动的方式2
    爹没有高度，儿子全部浮动，爹就死，找不到高度。这样的话，这个家族会对下一个家族产生影响。我们的网页将无法布局。这会我们要去清除浮动。目的是既能让浮动的儿子撑出爹的高度，也能让爹找到自己的高度。到家里的最后放个空盒子，对这个空盒子添加clear:both;
  • 清除浮动的方式3
    爹没有高度，儿子全都浮动，爹就死，找不到高度，。这样的话，这个家族会对下一个家族产生影响。我们的网页将无法布局。这会我们要去清除浮动。目的是既能让浮动的儿子撑出爹的高度，也能让爹找到自己的高度。对爹添加属性overflow:hidden;

二、超级链接的美化

1、我们在玩a的时候，把盒子的信息和样式写在a标签当中，其他的文本类的颜色。背景等样式，用伪类来写
2、当我们写伪类的时候伪类遵循顺序来写。
link visited hover activie我们叫先爱后恨
3、当我们写完伪类的时候，如果每个伪类的前两个样式一样，那么他们可以写在a标签当中
4、a标签在实际项目中，通常用的是伪类就是hover，active很少用的，基本不用，我们以后去设置a的时候，一般都是点击前和点击后的样式相同，悬浮的时候是另外一种样式。所以，以后用的最多的就是link visite hover 而link和visite可以合并写在a中。

三、定位

position: static relative absolute fixed
分别是静态 相对 绝对 固定
static：和默认是一样的。
relative：相对定位，相对自身默认出现的位置去做定位

没有脱离标准文档流。用的很少，在后面子绝父相的时候用来做参照，形影分离；

absolute：绝对定位，相对自身最近的具有定位属性的祖先进行定位，如果所有祖先都没有定位，那么他就相对body定位。脱离标准文档流。

fixed：固定定位，相对浏览器窗口进行定位。脱离标准文档流。

定位的元素都存在一个属性叫做z-index
这个属性值控制谁该压着谁
一般都是同一个家族，几个儿子之间取值控制z-index
谁压着谁这个问题，我们分以下几种情况
1、同一个家族，定位的儿子永远压着没定位的儿子
2、同一个家族，所有的定位的儿子都没有设置z-index值，那么谁在后面谁就在最上面
3、同一个家族，所有的定位的儿子都设置z-index值，谁的z-index值大，谁就在最上面。
还有一种情况不多讲但是也会碰到，不同的家族，谁该压着谁，现在有两个家族都分别有儿子，
如果这个两个家族的爹都不设置定位，那么它们的儿子都相对于body定位。它就和一个家族的儿子一样；
如果这两个家族的爹都设置了定位，那么它们的儿子谁压着谁，要看爹有没有z-index值，如果它们的爹的z-index值相同，儿子的z-index值也相同，那么后面的家族永远压着前面的家族，如果爹的z-index值不同，那么先看爹，爹怂儿子再也NB不了了；
一句话不同的家族就是拼爹，如果爹的z-index值相同，在拼自己。
