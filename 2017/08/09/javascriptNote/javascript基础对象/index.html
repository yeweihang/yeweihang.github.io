javaScript基础对象

js的String对象

  • 创建String对象

    var str = “string”;

方法和属性

属性

  • length：字符串长度

    方法

  • 与HTML相关的方法

    bold();加粗
    fontcolor(color);设置颜色
    fontsize();设置字体大小(1~7)
    link(url);将字符串显示为超链接
    sub();和sup();显示上标和下标

  • 与Java相似的方法

    concat(String1,String2…Stringn);连接字符串
    charAt();返回指定位置的字符
    indexOf();返回指定字符的位置
    splitsplit(separator, howmany);把字符串分割为字符数组
    replace(regexp,replacement);替换字符串
    subStr()和subString

js的全局函数

由于不属于任何一个对象，直接写名称使用

  • eval();执行js代码(如果字符串是一个js代码，使用方法直接执行)
  • encodeURI();对字符进行编码
  • decodeURI();对字符进行解码
  • encodeURIComponet();
  • decodeURIComponent();
  • isNaN();判断当前的字符串是否是数字(是数字返回false，不是数字返回true)
  • parseInt();

js函数的重载

什么是重载———方法名相同，参数不同
js的重载是否存在？不存在。调用最后一个方法，把传递的参数存在arguments;
但是可以通过其他方式实现重载

js的bom对象

bom: broswer object model:浏览器对象模型

  • navigator

    获取浏览器的信息
    navigator.appName;

  • screen

    获取屏幕信息

  • location

    请求URL地址
    herf属性

    • 获取请求的地址
    • 设置URL地址
      • 页面上安置一个按钮，按钮绑定一个事件，当我点击这个按钮，页面跳转到另外一个页面

  • window
    窗口对象
    顶层对象(所有的bom对象都是在Window里面操作)

    • alert();
    • confirm();确认提示框，返回true与false
    • prompt(text,defaultText);(现在很少用，太丑)输入对话框
    • open(URL,name,features,replace);打开一个新的窗口
      • URL：打开新窗口的URL地址
      • features：窗口特征，比如窗口的宽和高
      • close();关闭窗口
    • setInterval(code,millisec);周期执行代码
      • code:js代码
      • millisec：毫秒数
    • setTimeout(code,millisec);在毫秒数之后执行，只会执行一次
    • clearInterval();清除setInterval设置的定时器
    • clearTimeout();清除setTimeout设置的定时器
