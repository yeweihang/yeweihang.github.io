Java三大变量

  • 类变量（静态变量）
  • 实例变量
  • 局部变量（本地变量)
  • 类变量

    类变量是声明在类内，方法之外，且使用static修饰的变量

  • 实例变量

    实例变量是声明在类内，方法之外，且未使用static修饰的变量

  • 局部变量

    局部变量声明在方法、构造器或者块内，且在方法、构造器或者块进入时被创建，一旦退出该变量就会被销毁

3者的区别：

1、存储位置

静态变量存储于方法区
实例变量存储于堆区
局部变量存储在栈内存中

2、生命周期

静态变量在加载类过程中优先加载，其生命周期取决于类的生命周期
实例变量在创建实例时才创建，它的生命周期取决于实例的生命周期
局部变量在方法、构造器或者块进入时被创建，一旦退出该变量就会被销毁

3、引用对象

静态变量属于类，被类的所有实例共享，可以直接使用类名来引用也可以通过类的实例引用
实例变量属于某个对象，它必须在创建对象后才可以通过这个对象来使用
局部变量属于块内，且只在块中作用

4、默认值

静态变量有默认值，数字的默认为0，boolean的默认为false，对象引用默认值为null,且可以在特殊的静态初始化区赋值
实例变量有默认值，数字的默认为0，boolean的默认为false，对象引用默认值为null
局部变量没有默认值，因此本地变量必须被声明并且在第一次使用前要给它赋值

实例

运行

staticVar=2;instanceVar=2;chunk=2
staticVar=3;instanceVar=2;chunk=2
